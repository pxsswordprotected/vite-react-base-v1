# Project Name: boilerplate-base

## Guiding Principles
- **Design Philosophy:** Clean, functional, simple (but no more than simple) and honest. No "dark patterns" or over-engineering.
- **Communication:** Use straightforward language. No corporate jargon.

## Reference Material (The "Brain" folder)
- **Design Practices:** Refer to the notes in `boilerplate-mine\notes\gen` before building new stuff.
- **Animations:** Consult the notes in `boilerplate-mine\notes\animation` for animation notes to use to build animations.
